# Zephyrise 🌬️

**One Browser. Infinite Reach.**

Zephyrise is an AI-native browser designed to make browsing easier and smoother.

## 🌟 Vision
A browser that doesn’t just display the web — it understands it.

## ⚡ Principles
- Make everything easier
- Make everything smoother
- AI should assist, not interrupt
- Speed over complexity
- Technology should disappear into the background

## 🏗️ Stack
- Electron
- React
- TypeScript
- Vite
- pnpm (monorepo)

## 📁 Structure
Monorepo-based architecture using pnpm workspaces.

## 🚧 Status
Early development (v0.0.1)
